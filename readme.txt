== Description ==

WP MathJax is another simple wordpress plugin for MathJax. WP MathJax will add MathJax.js to the header of the page. MathJax.js can be either from Plugin's local MathJax or from MathJax CDN. The MathJax configuration can be easily change from Plugin's setting page. The available configuration options will be loaded from the local MathJax library.

== Installation ==

1. Extract the plugin archive
1. Upload plugin files to your /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configuration can be changed from WP MathJax menu from Settings
